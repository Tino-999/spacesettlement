\# Space Settlement Index



A minimal, monochrome, SpaceX-like visual index of \*\*people\*\*, \*\*projects\*\*, \*\*organizations\*\*, and \*\*topics\*\* advancing space settlement.



\- \*\*Landing\*\* (`index.html`): generative art + entry point  

\- \*\*Index\*\* (`library.html`): large image sections + short descriptions

\- Content lives in `data/items.json`

\- Images go in `assets/img/cards/...`



\## GitHub Pages

Deploy from the repository root (no build tools).  

Keep paths relative.



\## Add an item

1\. Add an image under `assets/img/cards/<category>/...`

2\. Add a new object in `data/items.json`

3\. Commit + push
